选项卡切换组件

- 何时使用

  • 提供平级的区域将大块内容进行收纳和展现，保持界面整洁
  • 标准线条式页签，用于容器内部的主功能切换，这是最常用的 Tabs
  • Buttons可作为更次级的页签来使用

- 参数

参数 说明 类型 默认值
defaultActive 默认激活的选项卡编号 Number 0

- 方法

方法 说明 类型
bootCollapse(Option) 接收一个可选的对象，初始化插件 Object
bootTab(Number) 指定激活对应编号选项卡 $("#tab").bootTab(0)
bootTab(String) 指定激活对应标题选项卡 $("#tab").bootTab('javascript')
