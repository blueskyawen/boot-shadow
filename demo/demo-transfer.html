Transfer穿梭框

双栏穿梭选择框

- 何时使用

  • 用直观的方式在两栏中移动元素，完成选择行为
  • 择一个或以上的选项后，点击对应的方向键，可以把选中的选项移动到另一栏

- 参数

参数 说明 类型 默认值
sourOptions 源数据项列表 Array []
targOptions 目标数据项列表 Array []
disableItems 禁止选择项 Array []
itemText 项目文本 String 项目
sourceText 源项目文本 String 源
targetText 目标项目文本 String 目标
height 框高度 Number 240
width 框宽度 Number 200

- 方法

方法 说明 类型
bootTransfer(Option) 接收一个可选的对象，初始化插件 Object
bootTransfer('getSource') 获取源框中的数据对象列表 -
bootTransfer('getTarget') 获取目标框中的数据对象列表 -
