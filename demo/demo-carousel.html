Carousel轮播

旋转木马，一组轮播的区域, 走马灯

- 何时使用

  • 当内容空间不足时，可以用走马灯的形式进行收纳，进行轮播展现
  • 常用于一组图片或卡片轮播

- 参数

参数 说明 类型 默认值
options 图片数据列表 Array []
type 类型（渐变或滑动）;fade/slide String fade
dataApi 是否支持data-api Boolean false
interval 自动播放的间隔时长 Numder 5000ms
pause 是否支持暂停播放 Boolean false
width 宽度 百分比 100%
height 高度 Numder -

- 方法

方法 说明 类型
bootCarousel(Option) 接收一个可选的对象，初始化插件 Object
bootCarousel(NUM) 指定显示某个图，入参为索引值 -
bootCarousel('prev'/'next') 指定显示上一个/下一个 -
bootCarousel('cycle') 恢复自动播放 -
bootCarousel('pause') 暂停自动播放 -
