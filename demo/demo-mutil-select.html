- 参数

参数 说明 类型 默认值
options 下拉项目数组 Array []
defalutSelected 默认选择的项目 Array []
disable 默认禁止的项目 Array []
disabled 是否禁止下拉选择 Boolean false
checkbox 是否携带复选框 Boolean false
allcheck 是否携带全选框 Boolean false
width 下拉框宽度 Number 420

方法 说明 类型
