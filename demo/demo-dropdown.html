Dropdown下拉菜单

弹出的菜单列表

- 何时使用

  • 当页面上的操作命令过多时，用此组件可以收纳操作元素
  • 点击或移入触点，会出现一个下拉菜单。可在列表中进行选择，并执行相应的命令

- 参数

参数 说明 类型 默认值
type 下拉菜单触发类型 String;click/hover click
place 下拉列表显示位置 String;left/right/down/up down
divi 是否分割触点 Boolean false

- 方法

方法 说明 类型
bootDropdown(Option) 接收一个可选的对象，初始化插件 Object
bootDropdown('toggle') 触发下拉列表，打开或关闭 -
